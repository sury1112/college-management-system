Machinist

Course Overview

To train candidates in operating machine tools (lathe, milling, grinding, drilling) to produce precision components, equipping them for roles as machinists, CNC operators, or tool setters in manufacturing, automotive, or aerospace industries, or for self-employment in machining workshops.

Course Contents

  • Introduction to Machining
  • Workshop Tools
  • Lathe Machine Basics
  • Cutting Tools
  • Engineering Drawing
  • Drawing Interpretation
